# Vikashvar Rajan Portfolio

A professional static portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Dark theme
- Smooth scrolling navigation
- Sections: About, Services, Tech Stack, Experience, Portfolio, Blog, Contact
- Download CV link

## Usage

1. Open `index.html` in your web browser.
2. Navigate through the sections using the menu or scrolling.

## Customization

- Update personal information in `index.html`
- Modify styles in `style.css`
- Add interactivity in `script.js`

## Troubleshooting

- Ensure all files are in the same directory.
- If images don't load, check internet connection for CDN links.
- For local development, use a local server if needed, but static files work directly in browser.
